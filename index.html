How to set up Google OAuth

One-time setup takes about 5 minutes. Your data stays in your own Google account.

Step-by-step setup
1
Go to console.cloud.google.com and create a new project (or use existing)
2
Search for "Google Sheets API" → click Enable
3
Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID
4
Application type: Web application. Add your website URL (or http://localhost for local use) to Authorized JavaScript Origins
5
Copy the Client ID (ends in .apps.googleusercontent.com) and paste it in the Connect tab
6
Go to APIs & Services → OAuth consent screen → Add your email as a test user if app is not published
💡For personal use, you don't need to publish the app. Just add yourself as a test user on the OAuth consent screen.
